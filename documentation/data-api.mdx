# Stablecoin Data API

`https://data.brale.xyz`

Public API for querying stablecoin metadata and contract addresses across multiple blockchains. No authentication required. CORS enabled for browser and agent use.

**Base URL**: `https://data.brale.xyz`  
**Auth**: None  
**Content-Type**: `application/json`  
**Cache**: 10 minutes (`max-age=600`)  
**Version**: 1.0.0 (released 2025-01-27)

---

## Supported Tokens

| Symbol | Name | Networks |
|--------|------|----------|
| `SBC` | Stable Coin | Ethereum, Polygon, Avalanche, Arbitrum, Celo, Optimism, Base, Solana, Stellar |
| `USDC` | USD Coin | Ethereum, Polygon, Avalanche, Arbitrum, Optimism, Base |
| `USDGLO` | Glo Dollar | Ethereum, Polygon, Base |
| `CFUSD` | Coinflow USD | Base, Polygon |

Symbols are **case-insensitive** — `sbc`, `SBC`, and `Sbc` all work.

### GET /price/{symbol}

Get the current price for a stablecoin. Stablecoins maintain a $1.00 USD peg.

**Parameters**

| Name | In | Required | Description |
|------|----|----------|-------------|
| `symbol` | path | Yes | Token symbol: `sbc`, `usdc`, `usdglo`, `cfusd` (case-insensitive) |

```bash
curl https://data.brale.xyz/price/sbc
```

```json title="Response (200)"
{
  "symbol": "SBC",
  "name": "Stable Coin",
  "price": "1.00",
  "currency": "USD",
  "details": "https://data.brale.xyz/data/sbc"
}
```

```json title="Error (404) — unknown symbol"
{
  "error": "No price found for ticker: FAKE123"
}
```

## Common Use Cases

### Look up a contract address

To find the SBC contract address on Base (chain ID `8453`):

```bash
curl -s https://data.brale.xyz/data/sbc | jq '.tokens[] | select(.chainId == 8453)'
```

```json
{
  "chainId": 8453,
  "address": "0xfdcC3dd6671eaB0709A4C0f3F53De9a333d80798",
  "name": "Stable Coin",
  "symbol": "SBC",
  "decimals": 18,
  "logoURI": "ipfs://Qmei6RQgkLNjS2FsTgryBZ41Z9dqMrRV4WMtEgffgjwiRs"
}
```

### List all networks for a token

```bash
curl -s https://data.brale.xyz/data/usdc | jq '[.tokens[].chainId]'
```

```json
[1, 137, 43114, 42161, 10, 8453]
```

### Discover available tokens

```bash
curl -s https://data.brale.xyz/list | jq '.tokens[].name'
```

```
"Brale SBC Token List"
"USDC List"
" USDGLO Token List"
"Coinflow USD"
```
