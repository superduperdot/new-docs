---
title: 'Burn tokens'
description: 'Burn (destroy) tokens from an address. Only available for tokenization accounts.'
openapi: 'POST /accounts/{account_id}/tokens/burns'
---
Use the playground below to try this endpoint directly, or review the OpenAPI details in the right panel.

<Warning>
Requires `self_attested_tokens:burn` scope. Only available for tokenization/self-attested accounts. Requires `Idempotency-Key` header.
</Warning>
