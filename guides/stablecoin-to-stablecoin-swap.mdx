<p class="intro-subheading">
  Brale makes it easy to swap between stablecoins, and to swap stablecoins between blockchains. See
  Supported Stablecoins and Blockchains for the full list.
</p>

## Prerequisites

- Bearer token ([How to authenticate](/key-concepts/authentication))
- Your `account_id` ([How to get your account_id](/guides/how-to/get-your-account-id))
- An internal (custodial) `address_id` with a non-zero balance ([How to get address_ids](/guides/how-to/get-your-address-ids))

## Swap Your Stablecoin Cross Chain

Stablecoins are issued natively on each chain and can be swapped 1:1 between chains at any time. Stablecoins are burned on the source chain, and minted on the destination chain.

### Swap from Solana to Base

**POST** `https://api.brale.xyz/accounts/{account_id}/transfers`

```bash
curl --request POST \
  --url "https://api.brale.xyz/accounts/${ACCOUNT_ID}/transfers" \
  --header "Authorization: Bearer ${ACCESS_TOKEN}" \
  --header "Content-Type: application/json" \
  --header "Idempotency-Key: $(uuidgen)" \
  --data '{
    "amount": {
      "value": "100",
      "currency": "USD"
    },
    "source": {
      "address_id": "'${SOURCE_ADDRESS_ID}'",
      "value_type": "SBC",
      "transfer_type": "solana"
    },
    "destination": {
      "address_id": "'${DEST_ADDRESS_ID}'",
      "value_type": "SBC",
      "transfer_type": "base"
    }
  }'
```

## Swap between Your Stablecoin and USDC

Swap between Brale-issued stablecoins, USDC, and Paxos-issued stablecoins 1:1 without slippage. Both source and destination can use the same `address_id` if they share the same chain.

**POST** `https://api.brale.xyz/accounts/{account_id}/transfers`

```json title="Request"
{
  "amount": {
    "value": "100",
    "currency": "USD"
  },
  "source": {
    "address_id": "${SOURCE_ADDRESS_ID}",
    "value_type": "USDC",
    "transfer_type": "solana"
  },
  "destination": {
    "address_id": "${DEST_ADDRESS_ID}",
    "value_type": "SBC",
    "transfer_type": "solana"
  }
}
```

## Verification

```bash
# Check balance before swap
curl -s "https://api.brale.xyz/accounts/${ACCOUNT_ID}/addresses/${ADDRESS_ID}/balance?transfer_type=solana&value_type=SBC" \
  --header "Authorization: Bearer ${ACCESS_TOKEN}"

# Create cross-chain swap — expect HTTP 201
curl -s -w "\nHTTP: %{http_code}" --request POST \
  --url "https://api.brale.xyz/accounts/${ACCOUNT_ID}/transfers" \
  --header "Authorization: Bearer ${ACCESS_TOKEN}" \
  --header "Content-Type: application/json" \
  --header "Idempotency-Key: $(uuidgen)" \
  --data '{
    "amount": {"value": "1", "currency": "USD"},
    "source": {"address_id": "'${SOURCE_ADDRESS_ID}'", "value_type": "SBC", "transfer_type": "solana"},
    "destination": {"address_id": "'${DEST_ADDRESS_ID}'", "value_type": "SBC", "transfer_type": "base"}
  }'

# Poll transfer status — expect status to transition from "pending" to "complete"
curl -s "https://api.brale.xyz/accounts/${ACCOUNT_ID}/transfers/${TRANSFER_ID}" \
  --header "Authorization: Bearer ${ACCESS_TOKEN}"
```

**Expected**: HTTP 201 with transfer object. Status transitions: `pending` → `processing` → `complete`.
