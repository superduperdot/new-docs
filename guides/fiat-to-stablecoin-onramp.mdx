title: 'Fiat to Stablecoin (Onramp)'
---

<p className="intro-subheading">
  Brale makes it easy for you and your customers to convert USD into stablecoins via wire transfer
  or ACH.
</p>

There are two ways to onramp to stablecoins through the Brale API:

- Submit a [Transfer](/key-concepts/transfers) request with a source value_type of `wire`, `ach_debit`, or `same_day_ach_debit`
- Create an [Automation](/key-concepts/automations), which automatically mints stablecoins to a pre-configured destination wallet when funds are received (Automatic Onramp)

Businesses can also onramp directly via the Brale Dashboard, and individuals can onramp through [Brale.io](https://brale.io/).

## Fiat to Stablecoin via Wire

Here is an example of onramping to your own stablecoin (YSBC) via wire transfer by calling the Transfers endpoint with the source `transfer_type` of `wire`.

**POST** `https://api.brale.xyz/accounts/account_id/transfers`

```json title="Request"
{
  "amount": {
    "value": "100",
    "currency": "USD"
  },
  "source": {
    "value_type": "USD",
    "transfer_type": "wire"
  },
  "destination": {
    "address_id": "2VcUIonJeVQzFoBuC7LdFT0dRe4",
    "value_type": "YSBC",
    "transfer_type": "solana"
  }
}
```

```json title="Response"
{
  "id": "transfer_123",
  "status": "pending",
  "amount": {
    "value": "100",
    "currency": "USD"
  },
  "source": {
    "transfer_type": "wire",
    "value_type": "USD"
  },
  "destination": {
    "transfer_type": "polygon",
    "value_type": "YSBC",
    "address_id": "2VcUIonJeVQzFoBuC7LdFT0dRe4"
  },
  "wire_instructions": {
    // ...Brale bank details
  }
}
```

Initially, the status of the transfer will be `pending`. Once Brale receives the funds at the bank account provided in `source_deposit_instructions` the stablecoins will immediately be minted to the destination wallet.

## Fiat to Stablecoin via ACH Debit

You can initiate ACH debits (same day or standard) from Plaid-linked addresses. See [Addresses](https://docs.brale.xyz/key-concepts/addresses) for the full flow of linking Plaid accounts as addresses you can pull from.

**POST** `https://api.brale.xyz/accounts/account_id/transfers`

```json title="Request"
{
  "amount": {
    "value": "100",
    "currency": "USD"
  },
  "source": {
    "address_id": "2VcUIonJeVQzFoBuC7LdFT0dRe4",
    "value_type": "USD",
    "transfer_type": "ach_debit"
  },
  "destination": {
    "address_id": "2VcUIonJeVQzFoBuC7LdFT0dRe4",
    "value_type": "SBC",
    "transfer_type": "canton"
  }
}
```

```json title="Response"
{
  "id": "transfer_123",
  "status": "pending",
  "amount": {
    "value": "100",
    "currency": "USD"
  },
  "source": {
    "address_id": "2VcUIonJeVQzFoBuC7LdFT0dRe4",
    "value_type": "USD"
    "transfer_type": "ach_debit",
  },
  "destination": {
    "address_id": "2VcUIonJeVQzFoBuC7LdFT0dRe4"
    "value_type": "SBC"
    "transfer_type": "canton"
  }
}
```

## Fiat to Stablecoin Automation (Automatic Onramp)

<div className="mt-8">
  Automations provide a unique account number and routing number that accepts wire transfers and ACH
  deposits, and automatically mints stablecoins to a pre-defined wallet address. Businesses can use
  automations to direct unique pre-funding coordinates to their customers, which mint stablecoins in
  customers' wallets.
</div>

Here is an example of onramping to your own stablecoin (SBC) via wire transfer by creating a fiat-to-stablecoin automation.

**POST** `https://api.brale.xyz/accounts/account_id/automations`

```json title="Request"
{
  "name": "XYZ Onramp",
  "source": {
    "value_type": "USD"
  },
  "destination": {
    "address_id": "2MhCCIHulVdXrHiEuQDJvnKbSkl",
    "value_type": "SBC",
    "transfer_type": "Solana"
  }
}
```

Automations are opened in a pending state and automatically transition into processing. Once virtual accounts are created and the automation is ready to use, it will transition to a completed status.

When an automation becomes active, the source.funding_instructions object is populated with the virtual accountâ€™s bank details (routing number, account number, beneficiary name, etc.). These are the coordinates you share with your customer to fund the automation.

**GET** `https://api.brale.xyz/accounts/{account_id}/automations/{automation_id}`

```json title="Response"
{
  "id": "35kVg1llbsOBYvx9XKr9eOtsD9z",
  "name": "Customer XYZ Automation",
  "status": "active",
  "source": {

      "beneficiary_name": "Company ABC",
      "routing_number": "000111222",

  },
  "destination": {
    "transfer_type": "solana",

```

## Brale.io (Beta)

[Brale.io](https://brale.io/) is a hosted onramp you can direct your users to in order to acquire stablecoins you have created. Users can acquire your stablecoin via:

- USDC
- ACH (coming soon)

All KYC requirements are performed by Brale.io directly.

To get access to [Brale.io](https://brale.io/), please reach out to our team at support@brale.xyz.
